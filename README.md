# openbox-themes

#### INSTRUCTIONS

##### Install
- Just copy and past theme-name to ~/.themes or /usr/share/themes

##### Choosing theme
 - Obconf
 - Manual
    - Edit ~/.config/openbox/rc.xml and change: "\<theme>\<name><b>theme-name</b>\</name>..."
